## 🚀 安装步骤

### 步骤1：备份
在更新前，建议备份 `backend/public/` 目录

### 步骤2：覆盖
将 `update/` 目录下的所有文件复制到 `backend/public/` 目录

```bash
# Linux/Mac
cp -r update/* /path/to/backend/public/

# Windows
# 直接拖拽覆盖即可
```

## 🔧 常见问题

### Q: 页面显示异常
**A**: 清除浏览器缓存，强制刷新（Ctrl+F5）

### Q: 轮播图不显示
**A**: 检查图片URL是否可访问

### Q: 动画效果不流畅
**A**: 确保使用的是现代浏览器（Chrome/Firefox/Safari）
