## 🚀 宝塔面板部署步骤（简化版）

### 1. 上传项目

将 `chongai-tp8/backend/` 整个目录上传到宝塔的 `/www/wwwroot/`

### 2. 创建数据库

- 数据库名：`chongai`
- 用户名：`chongai` (或使用root)
- 密码：自定义

### 3. 导入数据

- 在宝塔数据库管理中导入 `sql/chongai.sql`

### 4. 配置数据库

编辑 `/www/wwwroot/backend/.env` 文件：

```env
APP_DEBUG = true

[DATABASE]
TYPE = mysql
HOSTNAME = 127.0.0.1
DATABASE = chongai
USERNAME = 你的数据库用户名
PASSWORD = 你的数据库密码
HOSTPORT = 3306
CHARSET = utf8mb4
PREFIX = chongai_
```

### 5. 创建网站

- 域名：填写你的域名
- 根目录：`/www/wwwroot/backend/public`
- PHP版本：8.0+
- 运行目录：`/`

### 6. 配置伪静态

在网站设置 -> 伪静态，选择 `thinkphp` 或手动添加：

```nginx
location / {
    if (!-e $request_filename) {
        rewrite ^(.*)$ /index.php?s=$1 last;
        break;
    }
}
```

### 7. 设置权限

```bash
chown -R www:www /www/wwwroot/backend/runtime
chmod -R 755 /www/wwwroot/backend/runtime
```

## 📞 常见问题

**Q: 首页显示"页面错误"**
A: 检查伪静态是否配置正确，runtime目录是否有写权限

**Q: API返回404**
A: 确保public目录下有.htaccess且伪静态已配置

**Q: 数据库连接失败**
A: 检查.env中的数据库配置是否正确
